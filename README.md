# yuque_sae

语雀导出以及格式化

## 功能

1. 导出语雀所有知识库文档
2. 将语雀换行符 `<br />` 转换为两个空格加两个回车
3. 删除所有的 HTML 标签
4. 下载图片并替换文档中的 `url` 为本地路径

## 使用

下载 [release](https://github.com/NPCDW/yuque_sae/releases) 程序与 `config.example.yml` 文件，并将其重命名为 `config.yml` 放在同一文件夹

按照自己的需求修改 `config.yml` 配置

打开命令行，在命令行中执行

```
yuque_sae.exe
```
